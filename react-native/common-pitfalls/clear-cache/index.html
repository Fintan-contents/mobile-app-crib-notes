対処方法​

エミュレータの再起動​

エミュレータとターミナルを終了し、再起動します。
再起動しでもエラーが発生する場合はキャッシュ削除を実行してください。

キャッシュ削除​

次のコマンドでキャッシュが削除されます。

npx expo start --clear
