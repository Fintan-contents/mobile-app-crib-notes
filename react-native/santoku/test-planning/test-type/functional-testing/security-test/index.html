セキュリティに関する観点

機密情報やプライバシー情報を暗号化またはマスキングされているかを確認するテスト観点をまとめています。 それぞれのテスト観点毎の検証方法は、テスト方法にまとめています。

セキュリティ情報のログ出力​

NO詳細観点補足テスト方法
1機密情報機密情報について、元の情報が特定できない状態になっていること-B
2機密情報クラウド同期されるか否かが仕様通りであることiCloudやGoogle DriveへのバックアップB
3プライバシー個人情報について、元の情報が特定できない状態になっていること-B
