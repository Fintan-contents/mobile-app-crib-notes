ディープリンクの作成

ディープリンクURLの作成にはReact Native Firebase Dynamic LinksのAPIを利用し、共有しやすいように短いDynamic Linkを使用します。

ディープリンクURLの構成ルール​

ディープリンクURLのドメインには、アプリケーションに関連付けたドメインを使用する必要があります。 またコンテンツリンクURL（link）は、以下のように構成を決めて信頼できるURLかどうかを検証します。

本アプリケーションではFireabseのpage.linkドメインを利用します。ディープリンクURLの構成ルールは以下のとおりです。

項目APIのパラメータ値本番の設定例
ディープリンクURLのプレフィックスdomainUriPrefixアプリケーションに関連付けたドメインを持つURIのプレフィックスhttps://santokuapp.page.link/
コンテンツリンクURLlinkscreenで画面、keyとvalueでパラメータを指定https://santokuapp.page.link/${screen}?{key}=${value}
iOSのフォールバックURLios.fallbackUrliOSアプリをインストールするときのURLApp StoreのアプリのURL
バンドルIDios.bundleIdアプリケーションのbundleIdjp.fintan.mobile.SantokuApp
androidのフォールバックURLandroid.fallbackUrlAndroidアプリをインストールするときのURLGoogle PlayのアプリのURL
パッケージ名android.packageNameAndroidのアプリケーションIDjp.fintan.mobile.SantokuApp
