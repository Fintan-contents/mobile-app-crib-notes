非機能要件

サンプルアプリケーションの非機能要件についてまとめたものです。
政府公開のデジタル・ガバメント推進標準ガイドラインの「第3編ITマネジメントー第5章　要件定義」にある「ウ　非機能要件の定義」を参考に、非機能要件を定義します。 ここで定義したものは、Apple Review GuidelineとGoogle ヘルプ(審査のためにアプリを準備する)で定められているルールを遵守しています。

このアプリの非機能要件を定義するにあたっての前提​

このアプリではモバイルアプリケーションの考え方・アーキテクチャ整備を優先しているためバックエンド実装を最小限に留めます。 非機能要件の事項としてはバックエンドを対象にしたり含めて要件を定義するものがありますが、上記理由によりモバイルアプリのみを対象に非機能要件を定義しています。
