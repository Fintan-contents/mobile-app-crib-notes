iOSアプリの配布用ビルドの設定

iOSアプリの配布用ビルドには、ビルド環境devかstgまたはprodを利用します。また、ビルドタイプはdevではDebug、stgとprodではReleaseを利用します。

注記

ビルド環境やビルドタイプなど、ビルド設定の概要について理解が必要な場合はビルドを参照してください。

ビルド環境devとstgとprodには、アプリの配布のために必要なビルドの設定が含まれています。そのため、リポジトリにコミットされているリソースだけでは不足しており、次のリソースを別途入手する必要があります。

  • 開発用証明書
  • 開発用プロビジョニングプロファイル

fastlaneやXcodeでアーカイブを作成する場合は、以下のリソースも入手する必要があります。

  • 配布用証明書
  • 配布用プロビジョニングプロファイル

入手した証明書とプロビジョニングプロファイルを、それぞれキーチェーンとXcodeにインストールしてください。
