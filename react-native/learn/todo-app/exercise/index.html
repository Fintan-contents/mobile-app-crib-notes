演習課題

ここまで出来たら、次の演習課題にチャレンジしてください。 サンプルおよびガイドはありません。今までの実装をヒントにできるはずです。

ユーザ登録画面の作成#

ログイン画面を参考に、ユーザ登録画面を作成してください。 その際、ユーザ登録画面のパスワードは4桁以上が必要というチェックを追加してください。

  • ユーザ登録画面

タスクの削除#

ToDoタスクの削除機能を追加してください。

note

ToDo一覧画面に削除ボタンを追加する方法が簡単です。 その場合、確認ダイアログとの組み合わせで考えてみてください。 もちろん、違った形で実装しても問題ありません。
