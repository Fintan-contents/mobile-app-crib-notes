---
title: ユースケースに関する観点
---

このアプリの利用シナリオをベースに網羅的なテストを実施します。
以下に挙げる観点を検証できるようにテストケースを作成し、テストを実施します。
それぞれのユースケース毎の検証方法は、[テスト方法](../functional-testing.mdx#テスト方法)にまとめています。

| NO | 詳細 | 観点 | 補足 | テスト方法 |
| --- | --- | --- | --- | --- |
| 1 | 事前条件 | 事前条件が満たされている場合、満たされていない場合のそれぞれで仕様通りに振る舞うこと | - | B |
| 2 | トリガー | ユースケースを起動する各トリガーに対して、仕様通りにユースケースを開始できること | - | B |
| 3 | 基本フロー | ユースケースの基本フローが正常に完了できること | - | B |
| 4 | 代替フロー | ユースケースの代替フローが正常に完了できること | - | B |
| 5 | 例外フロー | ユースケース中に例外が発生した場合、設計された例外フローに沿ってユースケースが事後条件を満たせず終了すること | - | B |
| 6 | 例外フロー | 例外フローから復帰したあと、基本フローもしくは代替フローを正常に完了できること | 復帰可能な例外フローに限る | B |
| 7 | 事後条件 | ユースケースの事後条件が満たされていること | 当日の時間割を開始して、このアプリの利用が開始できていること | B |
