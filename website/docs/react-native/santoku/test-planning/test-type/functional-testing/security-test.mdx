---
title: セキュリティに関する観点
---

機密情報やプライバシー情報を暗号化またはマスキングされているかを確認するテスト観点をまとめています。
それぞれのテスト観点毎の検証方法は、[テスト方法](../functional-testing.mdx#テスト方法)にまとめています。

## セキュリティ情報のログ出力

| NO | 詳細 | 観点 | 補足 | テスト方法 |
| --- | --- | --- | --- | --- |
| 1 | 機密情報 | 機密情報について、元の情報が特定できない状態になっていること | - | B |
| 2 | 機密情報 | クラウド同期されるか否かが仕様通りであること | iCloudやGoogle Driveへのバックアップ | B |
| 3 | プライバシー | 個人情報について、元の情報が特定できない状態になっていること | - | B |
