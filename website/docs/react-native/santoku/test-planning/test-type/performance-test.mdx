## 目的

性能テストでは、性能に関する事項の[応答時間について](../../requirements/non-functional/performance.mdx#応答時間について)と[CPU・メモリ使用率について](../../requirements/non-functional/performance.mdx#cpuメモリ使用率について)で定めた性能目標を満たしているかどうかを確認します。性能目標を満たしていない場合、必要に応じてチューニングを実施します。

## テスト観点

| NO | 詳細 | 観点 |
| :--- | :--- | :--- |
| 1 | 応答時間 | ページ読み込みなどレンダリングにかかる時間が目標値以内であること |
| 2 | 応答時間 | ページ内でのスライドアニメーション動作などにかかる時間が目標値以内であること |
| 3 | CPU・メモリ使用率 | 何かしらの操作によって急激に上昇し、以降低下しない事象が発生しないこと |
| 4 | CPU・メモリ使用率 | 操作中緩やかに上昇するが、低下することなく上昇し続け高止まりする事象が発生しないこと |

## テスト方法

Android、iOS／iPadOS毎に[テスト端末バリエーション](../../test-planning/test-variation.mdx)の中から最も性能が低い機種を選択し、実機でテストを実施します。
