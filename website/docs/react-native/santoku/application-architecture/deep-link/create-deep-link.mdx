---
title: ディープリンクの作成
---

ディープリンクURLの作成には[React Native Firebase Dynamic LinksのAPI](https://rnfirebase.io/dynamic-links/usage#create-a-link)を利用し、共有しやすいように[短いDynamic Link](https://firebase.google.com/docs/dynamic-links/android/create)を使用します。

### ディープリンクURLの構成

ディープリンクURLのドメインには、アプリケーションに関連付けたドメインを使用する必要があります。
またコンテンツリンクURL（`link`）は、以下のように構成を決めて信頼できるURLかどうかを検証します。

本アプリケーションではFireabseの`page.link`ドメインを利用するためディープリンクURLの構成は以下のとおりです。

|項目|APIのパラメータ|値|本番の設定例|
|:-------|:-|:--|:---|
|ディープリンクURLのプレフィックス|domainUriPrefix|アプリケーションに関連付けたドメインを持つURIのプレフィックス|`https://santokuapp.page.link/`|
|コンテンツリンクURL|link|`screen`で画面、`key`と`value`でパラメータを指定|`https://santokuapp.page.link/${screen}?{key}=${value}`|
|iOSのフォールバックURL|ios.fallbackUrl|iOSアプリをインストールするときのURL|App StoreのアプリのURL|
|バンドルID|ios.bundleId|アプリケーションのbundleId|`jp.fintan.mobile.SantokuApp`|
|androidのフォールバックURL|android.fallbackUrl|AndroidアプリをインストールするときのURL|Google PlayのアプリのURL|
|パッケージ名|android.packageName|AndroidのアプリケーションID|`jp.fintan.mobile.SantokuApp`|
