## 管理対象のメッセージと管理対象外のメッセージ

管理対象とするメッセージを使用パターン毎に分類したリストを次に示します。

- 画面の表示項目
  - ラベル
  - 画面タイトル
  - 入力テキスト内のプレースホルダー
  - 操作ガイダンス
  - 入力エラーメッセージ
    - 単項目エラー
    - 相関項目エラー
- ダイアログ
  - 確認メッセージ
- スナックバー（※1）
  - 情報メッセージ
  - 警告メッセージ
  - エラーメッセージ

（※1）APIのレスポンス結果に応じて、モバイル端末側で用意したメッセージを表示します。ただし、システムメンテナンス時に迅速かつ適切な内容をユーザに伝えるため、APIレスポンスに含まれるメッセージをそのまま表示する場合もあります。

一方、管理対象外とするメッセージは次の通りです。

- ログ
  - ログメッセージ
- アクセス許可ダイアログ
  - カメラやマイク、位置情報などを利用する際に表示するアクセス許可ダイアログのメッセージ
