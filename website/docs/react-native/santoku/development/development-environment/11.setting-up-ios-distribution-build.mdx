---
title: iOSアプリの配布用ビルドの設定
---

iOSアプリの配布用ビルドには、ビルドタイプ`Release`と`ReleaseInHouse`を利用します。

:::note
ビルドタイプなど、ビルド設定の概要について理解が必要な場合は[ビルド](../build-configuration.mdx)を参照してください。
:::

ビルドタイプ`Release`と`ReleaseInHouse`には、アプリの配布のために必要なビルドの設定が含まれています。そのため、リポジトリにコミットされているリソースだけでは不足しており、別途入手する必要があります。

* 開発用証明書
* 開発用プロビジョニングプロファイル

fastlaneやXcodeでアーカイブを作成する場合は、以下のリソースも入手する必要があります。

* 配布用証明書
* 配布用プロビジョニングプロファイル

入手した証明書とプロビジョニングプロファイルを、それぞれキーチェーンとXcodeにインストールしてください。
