---
title: その他のツール
---

これまで紹介したツール以外に、よく利用されているツールを紹介します。どれも便利なツールなので利用してみてください。

| ツール                                                                                                           | OS             | 概要                                                                    |
|:----------------------------------------------------------------------------------------------------------------|:---------------|:-----------------------------------------------------------------------|
| [iTerm2](https://iterm2.com/)                                                                                   | macOS          | macOSのターミナルの代替                                                   |
| [Windows Terminal](https://www.microsoft.com/ja-jp/p/windows-terminal/9n0dx20hk701?activetab=pivot:overviewtab) | Windows        | Windowsで使えるターミナル                                                 |
| [Homebrew](https://brew.sh/index)                                                                               | macOS          | macOS用のパッケージマネージャで、様々なアプリをコマンドラインでインストールできる   |
| [Scoop](https://scoop.sh/)                                                                                      | Windows        | Windows用のパッケージマネージャで、様々なアプリをコマンドラインでインストールできる |
| [fish shell](https://fishshell.com/)                                                                            | macOS          | bashよりも補完や履歴検索が強力なシェル                                       |
| [asdf-vm](https://asdf-vm.com/)                                                                                 | macOS          | ディレクトリごとに開発ツールのバージョンを切り替えられるようにする                |
| [ghq](https://github.com/x-motemen/ghq)                                                                         | macOS, Windows | Gitリポジトリをクローンするときに、決まったディレクトリにクローンする             |
| [fzf](https://github.com/junegunn/fzf)                                                                          | macOS, Windows | コマンドラインで絞り込みを行うツールで、ghqなどと組み合わせると便利               |
| [bat](https://github.com/sharkdp/bat)                                                                           | macOS, Windows | シンタックスハイライトしてくれて、Windowsでも使える`less`コマンド               |
| [ripgrep](https://github.com/BurntSushi/ripgrep)                                                                | macOS, Windows | 高速かつ高機能で、Windowsでも使える`grep`コマンド                           |
| [xcpretty](https://github.com/xcpretty/xcpretty)                                                                | macOS          | `xcodebuild`の結果を見やすくフォーマット                                    |
