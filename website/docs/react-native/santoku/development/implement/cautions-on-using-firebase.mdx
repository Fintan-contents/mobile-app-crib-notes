---
title: Firebaseの機能を利用する際の注意点
---

このアプリでは、[ビルド - Firebaseの設定ファイル](../build-configuration/firebase-setting-file)に記載されている設定をすることで、Firebaseの機能を利用できます。

ただし、いくつか注意点があるため、ここではそれらについて説明します。

### Firebaseの設定ファイルがダミーの場合はFirebaseの機能を利用できません

Firebaseの設定ファイルは、ビルド環境が`local`の場合にダミーの設定ファイルが使用されます。

ダミーの設定ファイルが使用された場合は、Firebaseの機能を利用できません。

### ビルドタイプが`Debug`の場合は、アプリクラッシュ時にFirebase Crashlyticsにログを送信できません

ビルドタイプが`Debug`の場合は、React Nativeがアプリで発生したエラーをインターセプトして[LogBox](https://reactnative.dev/docs/debugging#logbox)を表示します。React Nativeがエラーをインターセプトするため、Firebase Crashlytics SDKではエラーを捕捉できず、Firebase Crashlyticsにはログが送信されません。
