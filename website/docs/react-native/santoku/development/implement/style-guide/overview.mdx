アプリ開発では、コーディング規約を定めることで一貫性を保ち、一定の品質を保証できます。

このアプリでは、ESLintなどの静的解析ツールによる自動チェックと、目視でチェックする規約を定めています。

## 静的解析ツール

ESLintやPrettier、TypeScriptなどの静的解析ツールを導入します。

以下のページでは、これらのツールの設定内容や運用方法について記載します。
