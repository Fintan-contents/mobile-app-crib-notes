このアプリで表示するメッセージの一覧を記載します。<br />
メッセージの種類とキーについては[メッセージの管理](/react-native/santoku/development/design/message-configuration.mdx)を参照。

## ラベル

| メッセージキー              | メッセージ                                                                     |
|:--------------------------|:-----------------------------------------------------------------------------|
| 利用規約                   | 利用規約                                                                       |
| 同意                       | 同意                                                                          |
| リロード                    | リロード                                                                      |
| 閉じる                    | 閉じる                                                                      |

## メッセージ

| メッセージキー         | メッセージ                                                                 |
|:--------------------|:------------------------------------------------------------------------|
| `app.初期化エラー`       | アプリの起動に失敗しました。アプリを再起動してください。                          |
| `app.ネットワークエラー` | 通信エラーが発生し、{0}に失敗しました。ネットワーク環境を確認してください。  |
| `app.terms.同意エラー`  | 利用規約の同意に失敗しました。ネットワーク環境を確認し、再度お試しください。 |
| `app.webview.onError`  | 通信エラーが発生し、コンテンツの取得に失敗しました。ネットワーク環境を確認してください。  |
