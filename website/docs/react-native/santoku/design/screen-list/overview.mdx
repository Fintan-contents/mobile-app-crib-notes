各画面の「概要／イベント／表示項目」を記載します。

## アプリ起動

### スプラッシュスクリーン

アプリケーション起動中にアプリのイメージを表示します。
アカウントの状態、起動パターンによって遷移先の画面が変わります。

- イベント
   - 次の画面へ遷移

- 表示項目
   - SplashScreen画像

## 利用規約

### 利用規約同意

ユーザに利用規約を見せ、同意を求める画面です。
アプリの利用には、ユーザの利用規約同意が必要です。

- イベント
  - 利用規約の同意／拒否

- 表示項目
  - 利用規約（WebView）
  - 同意ボタン
  - 拒否ボタン

## フッター

表示項目に記載のタブを切り替えるための、タブバーを表示します。
タブを切り替える時、それぞれのタブで表示している画面を維持します。（タブを切り替えてから戻ってきても、タブの初期表示画面には遷移しません）
