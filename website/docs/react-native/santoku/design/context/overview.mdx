## コンテキストの定義

このドキュメントでは以下のコンテキストについて記載します。

- 機能内コンテキスト
  - 特定の機能内において共有されるデータや状態を管理するコンテキスト
  - 例： 複数の画面にまたがって順次必要事項を入力する、ウィザード型入力における入力中の一時データ管理
- アプリ内共通コンテキスト
  - アプリ全体から参照されるデータや状態を管理するコンテキスト
  - 例： 現在の認証状態や、認証時に取得したアクセストークンやユーザ情報の管理
  - 例： 現在設定されている表示言語、タイムゾーン、カラーテーマの管理
