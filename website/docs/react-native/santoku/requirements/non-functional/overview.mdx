---
title: 非機能要件
sidebar_label: 概要
---

サンプルアプリケーションの非機能要件についてまとめたものです。  
政府公開の[デジタル・ガバメント推進標準ガイドライン][guideline]の「第3編ITマネジメントー第5章　要件定義」にある「ウ　非機能要件の定義」を参考に、非機能要件を定義します。
ここで定義したものは、[Apple Review Guideline](https://developer.apple.com/jp/app-store/review/guidelines/)と[Google ヘルプ(審査のためにアプリを準備する)](https://support.google.com/googleplay/android-developer/answer/9859455?hl=ja)で定められているルールを遵守しています。  

[guideline]:https://cio.go.jp/sites/default/files/uploads/documents/hyoujun_guideline_20210330.pdf
