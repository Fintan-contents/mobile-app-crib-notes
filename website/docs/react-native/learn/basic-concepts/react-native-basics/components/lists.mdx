リストはScrollViewに似ていますが、要素をリサイクルして再描画を減らすことでパフォーマンス向上するよう最適化されています。
そのため、`ScrollView` よりもやや複雑なコンポーネントとなります。

大量のデータを画面に表示する場合、`FlatList`または`SectionList`の使用が推奨されます。
