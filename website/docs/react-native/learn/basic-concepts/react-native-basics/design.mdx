React Nativeでは、JavaScriptを使用してアプリケーションのスタイルを定義します。
ほとんどのコアコンポーネントには`style` という名前のpropが用意されています。

スタイル名とその設定値は、キャメルケースを使用しているという点を除き、WebのCSSのそれと通常は一致します。
例えば、`background-color`は`backgroundColor`となります。
