React Nativeにおける最も基本的なコンポーネントです。
デフォルトでは何も表示せず、その他のコンポーネントのコンテナとしてスタイルやレイアウト用途として主に用いられます。
ターゲットプラットフォーム上の次のネイティブの部品に対応します。

- iOS - `UIView`
- Android - `android.view`
- Web - `<div>`
