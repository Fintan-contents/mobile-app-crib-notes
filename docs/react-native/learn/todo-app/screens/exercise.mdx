## ユーザ登録画面の作成

その際、ユーザ登録画面のパスワードは4桁以上が必要というチェックを追加してください。

- [ユーザ登録画面](../app-spec.md#ユーザ登録)

## タスクの削除

ToDoタスクの削除機能を追加してください。

:::note
ToDo一覧画面に削除ボタンを追加する方法が簡単です。
その場合、[確認ダイアログ](alert.mdx)との組み合わせで考えてみてください。
もちろん、違った形で実装しても問題ありません。
:::
