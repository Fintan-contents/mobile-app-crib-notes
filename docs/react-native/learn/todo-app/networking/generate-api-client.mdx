ToDoアプリでは各画面で`TodoService`からデータを取得しますが、`TodoService`から直接REST APIを呼びだすのではなく共通的なAPIクライアントを用意します。APIクライアントは、レスポンスのJSONとJavaScriptのクラスとの型変換や共通的なエラー処理を担当します。
